## 第一步：下载并安装 Clash Verge Rev

1. 前往 GitHub 的 Clash Verge Rev 官方开源仓库的 [Releases 页面](#)（国内网络可能无法直接打开，请使用上面的光速云临时代理访问）。
2. 在资源列表中，找到以 `.exe` 结尾的安装包（例如 `Clash.Verge_x.x.x_x64-setup.exe`）。
3. 下载完成后，双击运行。一路点击“下一步（Next）”直至安装完成。

*(此处应有一张清晰的安装包列表截图)*

## 第二步：获取并导入节点订阅链接

现在，我们需要把你在机场购买的节点导入到客户端中。

1. 登录你的机场后台（以光速云为例）。
2. 在“仪表盘”或“用户中心”首页，找到 **「一键订阅」** 或 **「导入到 Clash」** 按钮。
3. 点击 **「复制 Clash 订阅链接」**。

*(此处应有一张机场后台一键订阅按钮的截图)*

4. 打开刚刚安装好的 Clash Verge Rev 软件。
5. 在左侧菜单栏点击 **「配置 (Profiles)」**。
6. 在上方输入框内，**粘贴**你刚刚复制的订阅链接，然后点击旁边的 **「导入 (Import)」** 按钮。
7. 等待几秒钟，列表里会出现一个新配置文件。**右键点击它**，选择 **「使用 (Use)」**（此时它前面会亮起蓝色的小圆点）。

## 第三步：启动代理并测试网络

1. 点击左侧菜单栏的 **「代理 (Proxies)」**。在这里你可以看到所有的国家和地区节点（香港、日本、美国等）。
2. 在顶部选择 **「规则模式 (Rule)」**。这是最聪明的模式：访问国内网站（如淘宝、百度）不走代理，访问海外网站（如 Google、YouTube）自动走代理。
3. 展开“节点选择（PROXIES）”，点击一下右上角的 ⚡ **延迟测速图标**。几秒钟后，有绿色数字出现的节点就是可用节点。**选择一个延迟较低的节点（推荐香港或日本）**。

*(此处应有一张选中节点并测速成功的界面截图)*

4. 最后一步！点击左侧菜单栏的 **「设置 (Settings)」**，找到 **「系统代理 (System Proxy)」** 并将开关**打开**。

🎉 **恭喜你！** 现在打开浏览器，尝试访问 Google 或者 YouTube，你会发现新世界的大门已经敞开。
