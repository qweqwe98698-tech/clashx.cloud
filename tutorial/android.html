🚀 Clash Meta for Android 新手使用教程

💡 导语： 安卓手机端体验最好、开源且免费的 Clash 客户端，支持最新协议。
1

下载与安装软件

您可以前往 Github 发布页下载 Clash Meta for Android 的 APK 安装包并安装。

2

获取并导入订阅链接

登录您的机场后台，点击“一键订阅”，选择“复制 Clash 订阅链接”。然后打开 Clash Meta 客户端，点击底部的“配置”菜单，选择右上角的“+”号，选择“从 URL 下载”，将链接粘贴进去，名称随便填，点击保存。

3

选择节点与开启代理

回到“配置”列表，选中刚才下载的配置（前面的圈圈变亮）。然后回到主页（点底部的“代理”或“仪表盘”），点击顶部的“已停止”按钮将其启动。在代理界面选择一个低延迟节点即可。

❓ 常见问题排查 (FAQ)

1. 连不上怎么办？

请尝试更新订阅，并检查电脑/手机的时间是否准确（不能有误差）。

2. 国内网站变慢？

请确认客户端处于“规则/分流”模式，而不是全局模式。
