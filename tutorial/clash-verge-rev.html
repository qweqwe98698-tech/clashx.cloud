Clash Verge Rev 新手使用教程

💡 导语： Windows / macOS 平台最推荐的客户端，完美支持各大机场订阅。

1. 下载安装

前往 Github 下载最新版 Clash Verge Rev。

2. 导入订阅

在机场后台复制订阅链接，在软件的“订阅”界面粘贴并导入。

3. 开启代理

在设置中开启“系统代理”，在“代理”界面选择节点即可。
